### VSCodeによるLaTeX環境構築

本稿は全くの初心者が文書作成ツールである \LaTeX の環境構築を完了することを目的として作成する．まずはじめにTexWorksでの環境構築を行う．続いて，より便利な編集のためにVS Codeを用いて\LaTeX を使用する方法について解説を行う．最後に，\LaTeX 独自の書き方や執筆の上での注意事項について説明を行う．

.
├── LaTex_Manual.tex % マニュアルの.texファイル．どういうふうに書くとpdfが得られるのかを確認するのに使うことができる．
├── example.tex % サンプルコードの.texファイル．このファイルをそのままコンパイルすれば良い．
├── out % 出力を格納するディレクトリ
│   └── LaTex_Manual.pdf % 出力のpdf
├── readme.md % この説明書きのファイル
└── settings.json % VS Codeの設定ファイル．これをこのままコピーすれば良い．
